  # Terminal 1: El Cerebro (Backend)

   - cd server
   - npm run dev
  *Esto va a levantar el servidor en el puerto 3001 (o el que hayas puesto en el .env).*

  # Terminal 2: La Cara (Frontend)
   - cd client
   - npm run dev
  *Esto va a levantar Vite. Te va a dar una URL, que por defecto es http://localhost:5173.*
